Display : Flex (Child Property) Flex-Basis

  • flex-basis defines the initial size of a flex item before space is distributed by flex-grow or flex-shrink. It can be a length (e.g., 100px), auto, or 0, affecting how the item sizes within the flex container.
    • Flex-basis:Auto
    Flex-basis:20px
    Flex-basis:34%

    Display : Flex (Child Property) Flex-Grow

  • The flex-grow property defines how much a flex item will grow relative to the rest of the flex items inside a flex container.
  • It accepts a unitless number (e.g., 1, 2, etc.), where higher values mean the item grows more. If all items have flex-grow: 0, none will grow to fill extra space.
    • Flex-grow:Auto;
    Flex-grow:1;
    Flex-grow:2;

    Display : Flex (Child Property) Flex-Shrink

  • The flex-shrink property determines how much a flex item will shrink relative to others when there isn’t enough space. It uses a unitless number; higher values shrink more, while 0 prevents shrinking.
    • Flex-shrink:0;
    Flex-shrink:1;
    Flex-shrink:2;

    Display : Flex (Child Property) Align-Self:flex-start;

  • The align-self: flex-start property in Flexbox aligns a specific flex item to the start of the cross axis, overriding the container's align-items setting.
  • align-self: flex-start aligns a flex item to the start of the cross axis. It overrides the container’s align-items value for that item only.
