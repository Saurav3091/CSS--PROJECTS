Display : Flex (Parent Property) Justify-Content

  • justify-content is a CSS property that aligns and distributes items along the main axis in flex or grid containers. It controls spacing using values like flex-start, center, space-between, etc.
  • justify-content is a CSS property used in flexbox and grid layouts to define how space is distributed between and around items along the main axis of the container.

    • Display : Flex (Parent Property) Justify-Content : flex-start;

  • flex-start aligns flex items to the start of the main axis of the container.It places all items together at the beginning, without extra space between them.

    Display : Flex (Parent Property) Align-Items

  • Align-items is a CSS property used in Flexbox and Grid layouts to align items along the cross axis (perpendicular to the main axis).
  • It controls how items are positioned vertically (in a row layout) or horizontally (in a column layout).

    • Display : Flex (Parent Property) Align-Content : flex-start;

  • align-content: flex-start; is a CSS property used in Flexbox (when there are multiple lines of content). It aligns all the flex lines to the start of the cross axis, leaving extra space at the end.
